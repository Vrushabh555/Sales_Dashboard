# 📊 Sales Data Analysis Project (Excel + SQL + Power BI)

## 🔍 Project Overview
This project demonstrates end-to-end data analysis using Excel, SQL, and Power BI.

## 🎯 Objective
To analyze sales data and generate insights on performance, profit, and trends.

## 🛠 Tools Used
- Excel (Data Cleaning & Preparation)
- SQL (Data Analysis)
- Power BI (Dashboard & Visualization)

## ⚙️ Process
1. Data cleaning in Excel
2. Data analysis using SQL queries
3. Data visualization in Power BI

## 📊 Key Insights
- Identified top-performing regions
- Analyzed monthly sales trends
- Compared profit across categories

## 📁 Files Included
- sales_dashboard.pbix
- dataset.csv
- SQL_queries.sql

## 🚀 Conclusion
This project showcases skills in data cleaning, querying, and visualization.
